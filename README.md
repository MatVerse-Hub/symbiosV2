# symbiosV2

This repository is initialized and ready for development.

## Getting started

Clone this repository and enter the project directory:

```bash
git clone git@github.com:MatVerse-Hub/symbiosV2.git
cd symbiosV2
```

## First change in this cloned repository

After cloning, the repository is already initialized. Do not run `git init` inside the cloned directory. To make a first documentation change:

```bash
echo "Initial project note" >> NOTES.md
git add NOTES.md
git commit -m "Add initial project note"
```

## New repository example

Use `git init` only when starting a brand-new local repository that was not cloned from GitHub:

```bash
mkdir new-project
cd new-project
git init
echo "# new-project" > README.md
git add README.md
git commit -m "first commit"
```
